# CLI Note Manager

A command-line tool to manage notes with add, search, and delete functionality.

## Description
This project allows users to store notes locally and perform operations like viewing, searching, and deleting notes.

## Features
- Add notes
- View notes
- Search notes
- Delete notes
- File-based storage

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run:

   python notes.py

3. Choose options from menu
